# Behaviors Quick Reference

## One Mode + Any Qualities/Techniques

```
#=code #deep #tdd
```

---

## Modes

| Tag | Use When | Boundary |
|-----|----------|----------|
| `#=code` | Ready to implement | No unrequested features |
| `#=debug` | Something broken | No symptom patches |
| `#=review` | Code to evaluate | No implementation |
| `#=test` | Need to break it | No fixes |
| `#=spec` | Need a plan | No code |
| `#=research` | Need facts | No opinions |
| `#=assess` | Need interpretation | No action |
| `#=drive` | You steer, AI types | Small steps |
| `#=navigate` | AI steers, you type | No code |
| `#=record` | Document existing | No invention |
| `#=mentor` | Want to learn | No bare answers |
| `#=probe` | Think it through | No direct answers |

---

## Pipeline

```
research → assess → spec → code
```
