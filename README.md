# Nucleus

**Mathematical framework for AI prompting through symbolic equations**

```
engage nucleus:
[phi fractal euler tao pi mu] | [Δ λ ∞/0 | ε/φ Σ/μ c/h] | OODA
Human ⊗ AI
```

---

## The Core Idea

Replace verbose prompts with compressed mathematical symbols:

| Instead of... | Use... |
|---------------|--------|
| "Be thorough but efficient" | `[ε/φ Σ/μ c/h]` |
| "Check edge cases" | `[∞/0]` |
| "Think step by step" | `OODA` |

---

## How It Works

### Human Principles (What)

`[phi fractal euler tao pi mu]`

| Symbol | Property | Means |
|--------|----------|-------|
| **φ** | Golden ratio | Self-reference, natural proportions |
| **fractal** | Self-similarity | Scales predictably |
| **e** | Euler's number | Growth, compounding |
| **τ** | Tao | Minimal essence |
| **π** | Pi | Cycles, completeness |
| **μ** | Mu | Least fixed point, minimal |

### AI Principles (How)

`[Δ λ ∞/0 | ε/φ Σ/μ c/h]`

| Symbol | Means | Operation |
|--------|-------|-----------|
| **Δ** | Delta | Optimize via gradient descent |
| **λ** | Lambda | Pattern matching, abstraction |
| **∞/0** | Limits | Handle edge cases, boundaries |
| **ε/φ** | Approximate/Perfect | Tension: fast vs ideal |
| **Σ/μ** | Add/Minimize | Tension: features vs simplicity |
| **c/h** | Speed/Atomic | Tension: fast vs clean |

### Control Loops

| Loop | Means | Use For... |
|------|-------|------------|
| **OODA** | Observe → Orient → Decide → Act | Decision-making, debugging |
| **REPL** | Read → Eval → Print → Loop | Interactive development |
| **RGR** | Red → Green → Refactor | Test-driven development |

### Collaboration

| Operator | Behavior |
|----------|----------|
| **\|** | Parallel (equal partnership) |
| **⊗** | Tensor product (one-shot perfection) |
| **∘** | Composition (human bounds AI) |
| **∧** | Intersection (both must agree) |
